Code for SDN Infrastructure Deployment and Operability:

This section contains the codes required to automate and run the SDN environment for Team-06 Containerized Intelligent Network Functions on the Hosts project. The sections contain the following items:
1. Code to automate the deployment and connectivity of containerized OpenvSwitches to the host environment
2. Code to automate the deployment of Flows corresponding to VNFs on the OpenvSwitches on hosts
2. Python code for dynamic IP Address to MAC Address mapping and the periodic deployment of ARP entries on hosts
3. DNS code to verify URL requests as white list URLs or blacklist URLs
