Code for Testing Various Project Aspects:

This section contains the documentation required to test the functionality of the SDN environment. The sections contain the following items:
1. Main test code combines all the individual test cases into a single domain
2. Deployment Test verifies the deployment of SDN network infrastructure and network tools
3. Integration Test verifies the end to end connectivity between the SDN network elements
4. Compatibility Tests verifies the protocol level connectivity between SDN Controller and Hosts
5. Performance Test verifies the functionality of the service chain for different traffic types
6. Stress Test validates the redundancy in the solution
